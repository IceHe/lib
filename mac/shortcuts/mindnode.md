# MindNode

- https://mindnode.com/

## File

- `⌥ ↩` New Mind Map
- `⌥ →` Next Mind Map
- `⌥ ←` Prev Mind Map

## New

- `↩` Append New Sibling
- `⇧ ↩` Prepend New Sibling
- `⇥` New Child
- `⇧ ⇥` New Parent

## Edit

- `⌘ ↩` Edit Title ( Current )
- `⌘ ⇧ K` Note Popover
- `⌘ ⇧ T` Add / Rm Task ( Check Box )
- `⌘ ⇧ U` Toggle Task
- `⌘ k` Add Link
- `⌘ l` Add Connection ( to Another Node )

## Move

- `⌘ ↑` Move Above Sibling
- `⌘ ↓` Move Below Sibling
- `⌘ ⇧ D` Detach Node

## Fold

- `⌘ ⌥ ←` Fold Node
- `⌘ ⌥ →` Unfold Node
- `⌘ ⌥ ↑` Fold All Nodes
- `⌘ ⌥ ↓` Unfold All Nodes

## Select

- `⌘ ⌥ a` All Subnodes
