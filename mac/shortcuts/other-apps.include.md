#### cVim

Link

- `f` Open Link in Current Tab
- `F` Open Link in New Tab
- `mf` Open Links in New Tabs

Load

- `r` Refresh
- `R` Refresh ( including Cache )

Navigate

- `z` = `H` Backward
- `b` = `L` Forward
- `gu` Go up one path in the URL
- `gU` Go to the base URL
- `[[` Prev Page
- `]]` Next Page

Scroll Vertical

- `gg` to Top
- `G` = _`ge`_ to Btm
- `j ` = `凵` Dn a Half Page
- `k ` = `⇧ 凵` Up a Half Page

- `q` Up a Whole Page

Move Tab

- `W` = `<` to Lf
- `E` = `>` to Rg

Close / Reopen Tab

- `x` Close Current Tab
- `X` Reopen Last Closed Tab
