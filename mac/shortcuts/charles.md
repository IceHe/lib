# Charles

- https://www.charlesproxy.com/

## Proxy

- `⌘ r` [ Start | Stop ] Recording
- _`⌘ t` Start | Stop Throttling_
- _`⌘ k` Enable | Disable Breakpoints_
- _`⌘ ⇧ t` Throttle Settings_
- _`⌘ ⇧ k` Breakpoint Settings_
- _`⌘ ⇧ p` macOS Proxy_

## Session

- `⌘ ⌫` Clear
- _`⌘ o` Open_
- _`⌘ n` New_
- _`⌘ s` Save Request_
- _`⌘ ⇧ s` Save As …_
- _`⌘ l` Error Log_

## View

- `⌘ 1` Overview
- `⌘ 2` Request

- _`⌘ 0` Sequence_
- _`⌘ 9` Structure_
