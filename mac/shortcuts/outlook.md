# Outlook

- [Outlook](https://outlook.live.com/owa/)

## Usual

- `⌘ k` Send & Receive
- `⌘ r` Reply
- `⌘ ⇧ R` Reply All
- `⌘ j` Forward
- `⌘ ⇧ M` Move to Folder

## Edit

- `⌘ ⌥ R` Ribbon (Toggle Toolbar)
- `⌘ ⇧ V` Paste & Indent
- `^ ⌘ k` Hyperlink…

## Mark

- `⌘ ⇧ J` Mark as Junk
- `⌘ ⇧ T` Mark as Read
- `⌘ ⌥ t` Mark All as Read

## View

- `^ ⇧ [` Prev Pane
- `^ ⇧ ]` Next Pane

## Find

- `⌘ f` Find (current item)
- `⌘ ⌥ f` Advanced Find (current folder)
- `⌘ ⇧ F` Outlook Items

## Filter

- `⌘ ⇧ A` Has Attachment
- `⌘ ⇧ O` Unread
- `⌘ ⌥ o` Flagged
