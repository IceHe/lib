# File & Edit

## File

* `⌘ n` New
* `⌘ o` Open
* `⌘ s` Save
* `⌘ p` Print
* `⌘ ⇧ S` Save as

## Edit

* `⌘ z` Undo
* `⌘ ⇧ Z` Redo
* `⌘ a` Select All
* `⌘ c` Copy
* _`⌘ ⌥ c` Copy Path_
* `⌘ v` Paste
* `⌘ ⌥ v` Move \( After `⌘ c` \)
* `⌘ f` Search
* `⌘ g` Next Match
* `⌘ ⇧ G` Prev Match
* `⌥ ←` Prev Word
* `⌥ →` Next Word
* After text selection, then input:
  * `⌘ b` Bold
  * `⌘ u` Underline
  * `⌘ i` Italic
  * `⌘ +` Bigger \| Zoom In
  * `⌘ -` Smaller \| Zoom Out
