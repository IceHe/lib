- `` : Power

- `^` : Control | Ctrl
- `⇧` : Shift ( Shf )

- `凵` : Space
- `↑` : Up

- `←` : Left ( Lf )
- `→` : Right ( Rg )

- `0~9` : One of the digits 0 ~ 9
- `a~z` : One of the alphabet a ~ z
