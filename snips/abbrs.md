# Abbreviations

常见命名方式

- 保留头几个字母
- 去掉元音字母，留下有标识力的几个辅音字母
- ……

常见于类、方法、变量、命令等的命名…

## Symbols

- `&` : and
- `|` : or
- `@` : at
- `~` : home direcotry
- `*` : optional | wildcard character
- `?` : maybe | candidate
- `!` : not
- `>` : next

### Unicode

- `` : Power
- `⌘` : Command ( Cmd )
- `⌥` : Option | Alt
- `^` : Control | Ctrl
- `⇧` : Shift ( Shf )
- `⇪` : CapsLock
- `⇥` : Tab
- `⎋` : Escape ( Esc )

- `凵` : Space
- `↑` : Up
- `↓` : Down ( Dn )
- `←` : Left ( Lf )
- `→` : Right ( Rg )

- `0~9` : One of the digits 0 ~ 9
- `a~z` : One of the alphabet a ~ z
