### Unicode

- `` : Power
- `⌘` : Command ( Cmd )
- `⌥` : Option | Alt
- `^` : Control | Ctrl
- `⇧` : Shift ( Shf )
- `⇪` : CapsLock
- `⇥` : Tab
- `⎋` : Escape ( Esc )
- `↩` : Return ( Ret ) | Enter
- `凵` : Space
- `↑` : Up
- `↓` : Down ( Dn )
- `←` : Left ( Lf )
- `→` : Right ( Rg )
- `⌫` : Delete ( Del ) | Backspace
- `0~9` : One of the digits 0 ~ 9
- `a~z` : One of the alphabet a ~ z
