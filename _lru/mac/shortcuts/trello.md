# Trello

- https://trello.com/

## Navigate

- `↑ ↓ ← →` Navigate Cards
- `⇧ ?` Show Shortcuts Tips
- `/` Search Cards
- `b` Select Board
- `f` _Filter Menu_
- `x` _Clear All Filters_

## Cards

Common

- `n` New Card
- `↩` Open Card
- `⌘ ↩` Save Text
- `e` Quick Edit
- `t` Title
- `d` Due Date
- `l` Label
- `;` Toggle Label Name
- `⎋` Close Menu / Cancel Editing
- `,` `.` `<` `>` Move Card ( to Adjacent List )
- `c` Archive Card

Seldom

- _`凵` Assign Self_
- _`s` Subscribe_
- _`m` Add/Rm Members_
- _`@` Autocomplete Members_
- _`#` Autocomplete Labels_
- _`^` Autocomplete Position_
